# Studentu praktisko darbu saraksts
## Tēma: PDF dokumentu analīze ar Python

### 1. uzdevums: PDF faila atvēršana un struktūras izpēte
Mērķis: iepazīties ar PDF dokumenta ielādi Python vidē.

Darba uzdevumi:
- ielādēt PDF failu ar `PyPDF2`;
- noteikt lapu skaitu dokumentā;
- izdrukāt katras lapas numuru;
- pārbaudīt, vai no lapas ir iespējams iegūt tekstu.

Sagaidāmais rezultāts:
- students prot atvērt PDF failu un nolasīt tā pamatinformāciju.

### 2. uzdevums: Teksta ekstraktēšana no PDF
Mērķis: iegūt tekstu no digitāli ģenerēta PDF dokumenta.

Darba uzdevumi:
- izvilkt tekstu no vienas lapas;
- izvilkt tekstu no visa dokumenta;
- saglabāt rezultātu `.txt` failā;
- noteikt biežāk sastopamos vārdus vai atslēgvārdus.

Sagaidāmais rezultāts:
- students spēj iegūt un saglabāt PDF teksta saturu tālākai analīzei.

### 3. uzdevums: Tabulu ekstraktēšana ar `pdfplumber`
Mērķis: iemācīties iegūt tabulveida datus no PDF faila.

Darba uzdevumi:
- atvērt PDF dokumentu ar `pdfplumber`;
- atrast dokumentā tabulu;
- pārvērst tabulu sarakstā vai `pandas DataFrame`;
- eksportēt rezultātu uz `.csv` failu.

Sagaidāmais rezultāts:
- students spēj izvilkt tabulu no PDF un sagatavot to analīzei.

### 4. uzdevums: Teksta un tabulu kvalitātes pārbaude
Mērķis: izvērtēt iegūto datu kvalitāti.

Darba uzdevumi:
- salīdzināt PDF saturu ar ekstraktēto rezultātu;
- identificēt rindkopu, simbolu vai kolonnu kļūdas;
- iztīrīt liekās atstarpes un tukšās rindas;
- sakārtot kolonu nosaukumus.

Sagaidāmais rezultāts:
- students saprot, ka PDF ekstrakcijai bieži nepieciešama pēcapstrāde.

### 5. uzdevums: OCR pielietošana skenētam PDF
Mērķis: iepazīties ar OCR izmantošanu skenētu dokumentu apstrādē.

Darba uzdevumi:
- salīdzināt digitālu PDF un skenētu PDF;
- veikt OCR teksta atpazīšanu no skenēta dokumenta;
- novērtēt atpazītā teksta kvalitāti;
- noteikt biežākās OCR kļūdas.

Sagaidāmais rezultāts:
- students saprot, kad nepieciešams OCR un kādi ir tā ierobežojumi.

### 6. uzdevums: Batch apstrāde vairākiem PDF failiem
Mērķis: automatizēt vairāku PDF dokumentu apstrādi.

Darba uzdevumi:
- apstrādāt visus PDF failus mapē;
- katram failam iegūt lapu skaitu un tekstu vai tabulu;
- saglabāt rezultātus strukturētā formā;
- izveidot kopsavilkuma tabulu par visiem failiem.

Sagaidāmais rezultāts:
- students spēj automatizēt PDF failu apstrādi Python vidē.

### 7. uzdevums: Mini projekts
Mērķis: pielietot apgūtās prasmes pilnā darba plūsmā.

Darba uzdevumi:
- izvēlēties vienu vai vairākus PDF dokumentus;
- noteikt piemērotāko apstrādes pieeju;
- iegūt tekstu, tabulas vai OCR rezultātus;
- sagatavot īsu kopsavilkumu par iegūtajiem datiem.

Sagaidāmais rezultāts:
- students demonstrē spēju patstāvīgi apstrādāt PDF dokumentus un interpretēt rezultātus.

## Vērtēšanas ieteikumi
- korekta Python koda izmantošana;
- spēja izvēlēties piemērotu bibliotēku konkrētam PDF tipam;
- rezultātu kvalitāte un datu tīrīšana;
- spēja izskaidrot apstrādes procesu un konstatētās problēmas.
